# Персональный ИИ-астролог — Продуктовый документ

> Документ фиксирует бизнес-решения, принятые в ходе продуктового анализа.

---

## Концепция продукта

**Персональный ИИ-астролог в кармане.**

Сервис читает всех авторитетных российских астрологов вместо пользователя, синтезирует их прогнозы с учётом персонального натального чарта и каждое утро отправляет один точный совет на день. После получения гороскопа пользователь может задать любой уточняющий вопрос — как личному астрологу.

---

## Целевой клиент

**Домохозяйка, 28–50 лет, Россия/СНГ**

### Профиль
- Жизнь: дом, дети, муж, хозяйство
- Активно использует Telegram, VK, Instagram, WhatsApp
- Технически умеренно грамотна — смартфон да, сложные приложения нет

### Психография
- Ищет поддержку и подтверждение своих решений
- Нуждается в эмоциональной стабильности и ощущении контроля над днём
- Доверяет конкретным именам (российские астрологи), а не абстрактным алгоритмам
- Делится контентом, который отражает её внутренний мир

### Реальные вопросы, которые её волнуют
- "Сегодня хороший день, чтобы поговорить с мужем о деньгах?"
- "Стоит ли мне сейчас начинать новое дело?"
- "Почему у меня такие сложные отношения со свекровью — это моя карма?"
- "Мы с мужем совместимы? А с подругой?"

## Уникальное торговое предложение (УТП)

> **"Личный астролог, который знает твою карту, каждое утро присылает совет на день и всегда готов ответить на твой вопрос"**

### Три уровня УТП

1. **Агрегация** — собираем прогнозы нескольких авторитетных российских астрологов
2. **Синтез** — ИИ анализирует и объединяет в один персонализированный текст под конкретный натальный чарт
3. **Диалог** — после получения гороскопа можно задать любой вопрос, и ИИ ответит как личный астролог, знающий твою карту

### Ключевое отличие от конкурентов
- Существующие сервисы дают **информацию** (один источник, один знак зодиака)
- Мы даём **уверенность и направление** (несколько источников + персональный чарт + живой диалог)
- Западные аналоги (Co-Star, The Pattern) не работают с российскими авторитетными источниками

## v3: Совместимость и виральные механики

### Новый функционал

#### 1. Проверка совместимости
Пользователь вводит данные другого человека (дату рождения) и получает персональный отчёт совместимости:

- **Общая совместимость** — % и описание стихий (Огонь+Воздух = гармония, Огонь+Вода = пар)
- **Любовь и чувства** — эмоциональная совместимость, языки любви по знакам
- **Конфликтные зоны** — о чём будете спорить и как это решать
- **Совет астролога** — конкретная рекомендация для этой пары

#### 2. Типы совместимости (выбор пользователя)
- 💕 **Романтическая** — с мужем / партнёром / потенциальным спутником
- 👯 **Дружеская** — с подругой / коллегой
- 👨‍👩‍👧 **Семейная** — со свекровью / мамой / ребёнком

Тип влияет на тон и акценты в отчёте.

#### 3. "Круг близких" — астрологическая карта отношений
Пользователь может сохранить нескольких людей (муж, дети, подруги) и:
- Видеть совместимость со всеми сразу
- Получать советы типа: "Сегодня у Маши (Скорпион) трудный день — поддержи её"
- Получать уведомления в дни рождения близких с астрологическим подарочным советом

### Виральные механики v3

#### Механика 1: "Проверь нашу совместимость"
```
Пользователь A проверяет совместимость с подругой B
         ↓
Бот генерирует красивую карточку совместимости
         ↓
A отправляет карточку B через кнопку "Отправить ей результат"
         ↓
B получает: "Маша проверила вашу совместимость — 87%!
             Хочешь узнать подробности? Напиши /start"
         ↓
B заходит в бот → новый пользователь
```
**Почему работает**: любопытство + лесть ("кто-то думает обо мне") + конкретный % как крючок.

#### Механика 2: "Пригласи подругу — узнай совместимость бесплатно"
- Первая проверка совместимости — бесплатна
- Для второй: "Пригласи подругу — и получи безлимитные проверки навсегда"
- Реферальная ссылка: `t.me/DailyVanga_bot?start=ref_USERID`
- Когда подруга регистрируется — обе получают бонус

#### Механика 3: "Совместимость дня" в группе
- Бот можно добавить в групповой чат подруг
- Каждый день: "Сегодня идеальная пара в чате: @Маша (Рыбы) + @Катя (Рак) — 94%! 🔥"
- Вовлекает всю группу, провоцирует обсуждение

#### Механика 4: "Астро-подарок подруге"
- Кнопка: "Подари подруге персональный гороскоп"
- Пользователь вводит дату рождения подруги
- Бот отправляет подруге красивую карточку: "Маша подарила тебе персональный гороскоп от звёзд ✨"
- Подруга видит свой гороскоп → регистрируется для ежедневных обновлений

#### Механика 5: "Звёздный рейтинг совместимости"
- Пользователь вводит нескольких людей
- Бот составляет рейтинг: "Твой топ совместимости: 1. Катя 94%, 2. Муж 82%, 3. Свекровь 51%"
- Это провоцирует поделиться и обсудить

### User Journey v3

```
Пользователь A (уже зарегистрирован)
         ↓
Кнопка "Проверить совместимость" (после гороскопа)
         ↓
Выбор типа: 💕 Романтическая / 👯 Дружеская / 👨‍👩‍👧 Семейная
         ↓
Ввод даты рождения другого человека
         ↓
Claude AI генерирует персональный отчёт совместимости
         ↓
Красивая карточка с результатом
         ↓
Кнопки:
  📲 "Отправить ей/ему результат" → виральный loop
  ➕ "Добавить в круг близких"
  🔄 "Проверить с другим человеком"
```

## Вирусные механики (общие)

### 1. Красивая карточка (основной инструмент)
- Гороскоп оформляется как визуальная карточка
- Одно нажатие → сторис Instagram, WhatsApp-чат, Telegram подруге
- Внизу карточки: *"Получи свой → [ссылка]"*

### 2. Совместимость (двусторонний вирусный loop)
- Она вводит данные подруги → получает отчёт
- Подруга получает уведомление с результатом
- Подруга заходит из любопытства → loop замкнулся

### 3. "Сбылось" (социальное доказательство)
- Через несколько дней: *"Помнишь, мы предупреждали о...? Сбылось?"*
- При положительном ответе — предложение поделиться
